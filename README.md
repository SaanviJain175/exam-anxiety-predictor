# Exam Anxiety Level Predictor

## 📌 Overview
This project predicts student exam anxiety levels (Low, Medium, High) using machine learning and provides personalised AI-based wellness suggestions.

## 🛠 Tech Stack
Python · Scikit-learn · Flask · LLaMA 3 API

## ⚙️ Features
- Random Forest classification model
- AI-generated personalised recommendations
- Dynamic anxiety escalation near exam dates

## 📊 Output
- Predicts anxiety level based on input features
- Generates actionable mental wellness suggestions

## ▶️ How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Modify input values to test predictions
